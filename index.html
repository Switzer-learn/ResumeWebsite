Projects

Simple POS System

Description:

This web-based Point of Sale (POS) system is designed for restaurant operations. It facilitates all from customer ordering to payment handling, kitchen display, and admin dashboard. Built using React Js, Node Js, Express Js, PostgreSQL, Axios, Tailwind CSS, and a RESTful API.

Book Note Project

Description:

An app to track books you’ve read, write notes, summaries, and manage book covers. Initially created while practicing SQL with PostgreSQL and later updated using Supabase. Built with Node Js, Embedded Js, Express Js, Bootstrap, and PostgreSQL.

E-commerce Page Prototype

Description:

A showcase design for a simple e-commerce site that can be further enhanced. Demonstrates the use of the fakestoreapi.com API. Built with Embedded Js, Node Js, Express Js, Axios, and Bootstrap.

Sakura Spa Bwi

Description:

An integrated project featuring a Landing Page, POS System, and Admin Dashboard for a spa company. The POS system handles transactions, payments, and revenue tracking while the admin dashboard manages inventory, employee management, and scheduling.

Built with React and TypeScript for a modern, responsive UI enhanced by Tailwind CSS. Supabase powers authentication and database management, while Node.js supports backend integrations.
